## ⚙️ Kurulum

1. Projeyi klonlayın.

```bash
git clone https://github.com/yaren-bakar/TravelTripProje.git
```

2. Visual Studio ile açın.

3. SQL Server bağlantı bilgilerini **Web.config** dosyasında düzenleyin.

4. Veritabanını oluşturun.

5. Projeyi çalıştırın.
